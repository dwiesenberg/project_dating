# ROR-Project

A dating application powered by Ruby on Rails stack and socket.io

## Getting Started

### Prerequisities

```
rails
postgres
socket.io
```

### Installing

```
bundle install
rails s
npm install
server.js
```
## Built With

* Rails
* Ruby
* Postgres
* Materialize
* Node.js
* Socket.io
* Express
